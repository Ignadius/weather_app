# Real-Time Weather Dashboard

A lightweight, asynchronous JavaScript application that fetches and displays real-time weather data based on user input. This project demonstrates how to orchestrate multiple API calls, handle asynchronous state, and dynamically update the DOM with error-handling fallbacks.

---

## Tech Stack

* **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+)
* **APIs:** OpenWeatherMap (Geocoding API & Current Weather API)
* **Architecture:** Promise-based asynchronous functions (`async/await`)
